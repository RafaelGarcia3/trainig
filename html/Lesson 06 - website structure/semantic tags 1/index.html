Introduction

This is an example using the semantic tags in HTML. This page use the most common semantic tags that help us to have structure and organization.
A semantic element clearly describes its meaning to both the browser and the developer.

  • Non semantic elements like <div>, <span> tells nothing about its content.
  • Semantic elements like <img>, <article>, <figure> clearly defines it content.
